## 🖥️ System Requirements
- Windows 10 or 11 (64-bit)
- WebView2 Runtime (installed automatically if missing)
- Active Xtream Codes IPTV subscription credentials

---

## ℹ About ODIN TV
ODIN TV is a modern, premium IPTV player built using Tauri + Rust + React. Bring your own playlist details and enjoy instant, smooth streaming of live channels, movies, and series.
